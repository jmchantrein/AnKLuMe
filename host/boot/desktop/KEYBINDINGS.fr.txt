anklume Desktop — Raccourcis clavier (KDE Plasma)
==================================================

KDE Plasma :

  Raccourcis standard KDE/Plasma Wayland.
  Configuration systeme > Raccourcis pour personnaliser.

  Super + Entree        Ouvrir Konsole
  Alt + F4              Fermer la fenetre
  Super + Fleches       Fenetre en demi-ecran / maximiser
  Meta + Tab            Changement de fenetre
  Ctrl + F1-F4          Changer de bureau virtuel

Disposition clavier :

  Alt + Maj             Basculer FR (AZERTY) / US (QWERTY)

Commandes rapides :

  anklume --help        Afficher toutes les commandes
  anklume console       Ouvrir la console tmux par domaine
  anklume learn start   Labs en web avec terminal integre
