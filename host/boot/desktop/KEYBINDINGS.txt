anklume Desktop — Keyboard Shortcuts (KDE Plasma)
==================================================

KDE Plasma:

  Uses standard KDE/Plasma Wayland keybindings.
  System Settings > Shortcuts for customization.

  Super + Enter         Open Konsole terminal
  Alt + F4              Close window
  Super + Arrow keys    Snap window to half-screen / maximize
  Meta + Tab            Task switcher
  Ctrl + F1-F4          Switch virtual desktop

Keyboard layout:

  Alt + Shift           Toggle French (AZERTY) / US (QWERTY)

Quick commands:

  anklume --help        Show all commands
  anklume console       Open tmux domain console
  anklume learn start   Web-rendered labs with terminal
