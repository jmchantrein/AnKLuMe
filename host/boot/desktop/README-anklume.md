d'apprentissage.

## Commandes utiles / Useful commands

```bash
# Lancer la plateforme web / Start web platform
python3 /opt/anklume/scripts/platform_server.py --port 8890

# Configurer le stockage / Configure storage
sudo /opt/anklume/scripts/start.sh

# Aide CLI / CLI help
anklume --help
```
